**Title**: ECG Signal Analysis

**Objective**: Analyze ECG signals to extract heart rate and detect arrhythmias.

**Overview**:
1. **Preprocessing**: Normalize and filter ECG signals.
2. **Visualization**: Plot signals in time and frequency domains.
3. **Filtering**: Apply FIR and bandstop filters.
4. **Peak Detection**: Identify R-peaks using `findpeaks()` and calculate heart rate.
5. **Arrhythmia Detection**: Check for abnormal heart rates and peak distances.
6. **Result Presentation**: Display heart rate and arrhythmia status.

**Tools**: MATLAB with Signal Processing Toolbox.

**Outcome**: Provides insights into heart health and arrhythmias for medical diagnosis and research.
